# Прогнозирование месячной инфялции в России

## Автор
Рововой Валентин Владимирович

## Цель
Перед нами стоит задача в **создание модели, способоной предсказывать данные по месячной инфляции в России.**

В проекте были использованы следующие показатели, данные с 2000 по 2025 года: 
-Данные по месячной инфляции в России 
-Отношения рубля к доллару(USDRUB) 
-Цена фьчерса на нефть марки Brent 
-Значения индекса московской биржи (MOEX) 
-Данные индекса промышленного производства 
-Денежная масса М2 
-Индекс волатильности Российского рынка(RVI) 

## Стек технологий
-Pandas
-Matplotlib
-Seaborn
-Statsmodels
